# Information
The Locura T650 is now avalible with a Dump Bed. Load it up and just dump it out! Includes all the features you would expect from a Locura T-Series truck. Rear suspension is a live axle set up with quad offroad suspension on each main axle. Driver and passenger doors open as well as the hood.
